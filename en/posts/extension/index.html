Useful Browser Extensions

A curated list of browser extensions to block ads, analyze websites, collect SEO data, and work with fonts and colors.

May 31, 2025
3 minute read
Русский

1. AdGuard AdBlocker — Ad Blocking

In the age of information overload, distracting ads can disrupt your focus. A good ad blocker is essential — one of the best is AdGuard AdBlocker.

Installation link:
AdGuard AdBlocker on Chrome Web Store

Or search manually in your browser’s extension store:

  • Chrome, Opera, Edge: chromewebstore.google.com
  • Firefox: addons.mozilla.org
  • Search term: AdGuard AdBlocker

Configuration

  1. Click the extension icon → click the gear icon to open settings.
  2. In the Filters section, enable:
    • Ad Blocking
    • Privacy
    • Annoyances (blocks pop-ups like “Install Yandex Browser”)
    • Language-specific filters (e.g., Russian or your preferred language)
  3. Go to Tracking Protection to block trackers, including ad trackers.
  4. To manually block stubborn elements:
    • Right-click the extension icon → Block ads manually
    • Hover over the element and click to block.

P.S. I’m not against ads — but popups and phishing banners are just too much.

2. Wappalyzer — Website Technology Profiler

Wappalyzer is a handy tool to find out what technologies a website uses.

How to use:
After installation, click the extension icon to view data about the site’s CMS, analytics tools, plugins, and more (especially useful for WordPress sites).

3. Alaev SEO Tools — Express SEO Audit

Alaev SEO Tools provides a quick SEO overview of any webpage.

Features:

  • Overview — meta tags (title, description), headings H1–H6, sitemap status
  • Page — highlight H1–H6, check for missing alt tags, word count, reindexing in Yandex Webmaster, and PageSpeed testing
  • Links — view internal/external links and follow/nofollow attributes
  • Images — audit images for missing alt attributes
  • Structured Data — inspect OpenGraph and Twitter metadata

❗ This is a quick audit, not a full-scale analysis.
For full audits, use:

  • 🇷🇺 be1.ru, pr-cy.ru, rush-analytics.ru
  • 🌍 similarweb.com, semrush.com, ahrefs.com

4. Yandex Wordstat Assistant — Keyword Collection Tool

To collect keywords from Yandex Wordstat, use Yandex Wordstat Assistant.

Install from:

  • Chrome
  • Opera
  • Firefox

How it works:
Visit wordstat.yandex.ru — a panel will appear on the left side of the page.

Main functions:

  • Add keywords from the list
  • Copy keywords with or without frequency
  • Sort keywords alphabetically
  • Delete unnecessary queries

Copy your keyword list to Excel for analysis, clustering, and further SEO or ad campaign planning.

5. Font Finder — Inspect Web Fonts

Use Font Finder to inspect any text element on a webpage.

What you can view:

  • Font family
  • Size
  • Color
  • Styling properties (e.g., bold, italic)

Install from:

  • Chrome / Edge
  • Opera
  • Firefox

6. ColorZilla — Color Picker Tool

Pick any color from a webpage with ColorZilla. The hex code is automatically copied to the clipboard.

Extras:

  • Color Picker Panel — saves color history
  • Press Esc to deactivate the tool

Install from:

  • Chrome / Edge
  • Firefox
  • Opera

7. GoFullPage — Full Page Screenshots

GoFullPage – Full Page Screen Capture takes full-length screenshots of web pages.

How to use:

  1. Install the extension
  2. Click the icon to start capture
  3. Save the result as PNG or PDF

Install from:

  • Chrome / Edge
  • Not available for Firefox
  • In Opera, install as a Chrome extension
