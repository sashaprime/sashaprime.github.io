1. AdGuard Антибаннер — блокировка рекламы

В век информационной перегрузки реклама может сильно отвлекать от основного контента. В этом случае незаменим блокировщик рекламы. Один из лучших — AdGuard Антибаннер.

Ссылка для установки:
AdGuard Антибаннер в Chrome Web Store

Или найдите вручную в магазине расширений:

  • Chrome, Opera, Edge: chromewebstore.google.com/category/extensions
  • Firefox: addons.mozilla.org/en-US/firefox/extensions
  • Поиск: AdGuard Антибаннер

Настройка AdGuard

  1. Нажмите на иконку расширения → иконка «шестерёнка» → настройки.
  2. В разделе Фильтры включите:
    • Ad Blocking
    • Privacy
    • Annoyances (например, блокировка окон Яндекс.Браузера)
    • Языковой пакет: Русский или другой нужный
  3. Перейдите в раздел Tracking Protection — активируйте защиту от трекеров.
  4. Для ручной блокировки:
    • ПКМ по иконке → Block ads manually
    • Выделите элемент на странице — блокировка активируется.

P.S. Я не против рекламы, но попапы и фишинговые блоки — перебор.

2. Wappalyzer — определение технологий сайта

Расширение Wappalyzer – Technology Profiler показывает, какие технологии использует сайт.

Как пользоваться:
После установки нажмите на иконку — получите данные о CMS, аналитике и плагинах (особенно полезно для сайтов на WordPress).

3. Alaev SEO Tools — экспресс SEO-аудит

Alaev SEO Tools помогает быстро оценить SEO-состояние сайта.

Возможности:

  • Overview — title, description, заголовки H1–H6, карта сайта
  • Страница — alt-теги, количество слов, переиндексация, Pagespeed
  • Ссылки — follow/nofollow, внутренние/внешние
  • Изображения — наличие alt у всех картинок
  • Микроразметка — OpenGraph и Twitter metadata

❗ Это не полноценный аудит, а экспресс-анализ.
Для глубокой проверки используйте:

  • 🇷🇺 be1.ru, pr-cy.ru, rush-analytics.ru
  • 🌍 similarweb.com, semrush.com, ahrefs.com

4. Yandex Wordstat Assistant — сбор поисковых запросов

Для сбора ключевых слов из Wordstat используйте Yandex Wordstat Assistant.

Ссылка для установки:

  • Chrome
  • Opera
  • Firefox

Как пользоваться:
Перейдите на wordstat.yandex.ru — появится панель расширения.

Функции:

  • Добавление ключей
  • Копирование с/без частотности
  • Сортировка по алфавиту
  • Удаление ненужных запросов

Результаты можно сохранить в Excel для дальнейшей кластеризации и SEO-анализа.

5. Font Finder — шрифтовой инспектор

Наведите на текст и узнайте:

  • Цвет
  • Размер
  • Шрифтовое семейство
  • Стиль

Установка:

  • Chrome / Edge
  • Opera
  • Firefox

6. ColorZilla — пипетка и палитра

Выбирайте цвет любого элемента на странице. Цвет копируется в буфер обмена.

Дополнительно:

  • Color Picker Panel — история использованных цветов
  • Для выхода нажмите Esc

Ссылки для установки:

  • Chrome / Edge
  • Firefox
  • Opera

7. GoFullPage — скриншот всей страницы

GoFullPage – Full Page Screen Capture делает снимок всей страницы, а не только её видимой части.

Как пользоваться:

  1. Установите расширение
  2. Нажмите на иконку
  3. Сохраните результат в PNG или PDF

Ссылка для установки:

  • Chrome / Edge
  • Firefox — не поддерживается
  • В Opera можно установить как расширение для Chrome
